# Lost & Found Management System (MERN)

## 📌 Project Description
This is a MERN stack based Lost & Found system where users can:
- Register and Login
- Add lost/found items
- View all items
- Search items
- Delete items

## 🛠 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## 🔐 Features
- JWT Authentication
- Password hashing using bcrypt
- REST APIs
- Search functionality

## 🚀 How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start

## 🌐 API Endpoints
- POST /api/register
- POST /api/login
- GET /api/items
- POST /api/items
- DELETE /api/items/:id
- GET /api/items/search/:name
